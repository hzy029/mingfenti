# 正式题库蓝图

版本：v0.1  
日期：2026-05-01  
用途：在生成正式题干前，先固定 100 道题的结构、材料来源和测试意图。  
上游材料：`docs/详细设计/question-bank-material-map.md`

## 1. 使用规则

- 本文只定义题位，不写最终题干和 A/B/C/D 选项。
- 每题只有一个主分类，但可通过“材料卡片”承接交叉标签。
- `core` 题进入核心题库，后续权值建议为 2；`supplemental` 题进入补充题库，权值建议为 1。
- 生成正式题干前，涉及数字、引文和具体案例的题位必须回到材料卡片对应源文件核对。
- 样本类材料只用于选项话术，不作为题干事实依据。

## 6. 校对清单

- 核心题必须覆盖财政 8、军事 4、行政 5、横向 3。
- 补充题必须覆盖财政 30、军事 18、行政 20、横向 12。
- 同一材料卡片可以被多题引用，但同一题不得测试两个以上主要判断。
- 核心题优先选更有传播性、更能区分朱元璋梦男和理性历史观的题位。
- 补充题承担细分校准，不要求每题都非常尖锐。
- 生成题干前，所有涉及具体数字和引文的题位都回源核对。
