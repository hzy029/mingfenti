# 明粉话术总结

用途：给 `docs/详细设计/题库.md` 的 A/B 选项提供自然化话术参考。本文只总结话术类型，不把所有普通明代爱好者、普通汉服爱好者或普通清代研究者打成靶子。

## 1. 选项分层

| 选项 | 话术层级 | 写法目标 |
| --- | --- | --- |
| A | 强明粉 / 朱元璋梦男式话术 | 更冲、更嘴硬，优先护明朝形象；可以拐清、贴标签、护祖制、抬民族叙事或把网络梗当论据，但必须贴合题干。 |
| B | 缓和明粉 / 旧明粉式话术 | 有一点背景感，承认复杂性但仍倾向淡化明制责任；适合轻度偏明用户自然选择。 |
| C | 不了解 / 暂不判断 | 保留原来的“不确定、听过一点”功能。 |
| D | 材料与制度分析 | 保留原来的材料、机制、制度后果分析功能。 |

## 2. A 选项常用话术

- 拐清朝：提问明朝制度不利时，不正面处理明朝问题，转而说清朝也这样、清朝更差、清粉没资格批评。
- 清朝兜底：清朝更差，所以明代问题不值得追究；清朝类似问题更严重，所以批明是双标。
- 反明标签：批评明制就是明黑、满遗、清粉、宋粉或西方中心主义。
- 祖制护航：洪武祖制方向没错，错在后人执行坏了。
- 爱民神话：低税、实物财政、户籍控制、重典都可解释为乱世恢复秩序和爱民。
- 史料阴谋：明史被满清篡改，文官集团洗地，不利材料先怀疑立场。
- 人物滤镜：朱元璋、朱棣、左宗棠等人物魅力可以压过制度问题。
- 民族叙事压倒制度：汉人王朝、再造华夏、剃发易服等身份叙事优先于具体治理比较。
- 网络梗史料化：把土木堡在德国、抬明籍、明清粉互撕等梗当成可以认真护航的论据。

## 3. B 选项常用话术

- 否定存在：说问题本身不存在、被夸大、被后世黑明、被清修史料或清粉话术制造出来。
- 口径化解：承认表面现象，但说不能算失败、不能算弃地、不能算制度问题、不能用现代标准套。
- 局部化处理：把问题缩成个案、执行偏差、后期变坏或特殊时期，不承认祖制方向有问题。
- 初衷辩护：制度初衷有稳定秩序、节省成本、组织基层的意义，只是长期运行有代价。
- 背景复杂化：强调乱世、财政压力、边防压力或时代局限，降低明朝制度责任。
- 情感保留：可以讨论材料，但明朝身份、华夏叙事、人物情感仍有优先权。
- 梗与观点之间摇摆：承认某些说法像梗，但觉得背后可能有合理疑问。

## 4. 题库改写原则

- A/B 选项不能写成万能模板，必须回应题干中的具体对象，例如宝钞、卫所、汉服、土木堡、江南奴变。
- A 选项可以更冲、更粉圈、更嘴硬，但同一小节内不要连续使用同一种句式；拐清朝只是话术之一，不是每题默认答案。
- B 选项要像“懂一点背景但还想护”的真实回答，可承认现象、强调时代限制、局部执行、口径差异或初衷合理。
- 少用“这类问题”“口径误读”“局部执行问题”等万能套话；如果使用，必须落到本题材料上。
- A/B 都要有可选性，不能为了衬托 D 正确而故意写得像蠢答案。
- 不把普通兴趣误伤成极端立场；只有当兴趣被上升为朝代优越、族群斗争或制度护航时，才进入 A/B 话术。
- 样本材料只用作选项语气来源，不作为题干事实依据。
