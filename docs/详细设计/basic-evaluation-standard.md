# 普通版评测标准

版本：v0.1  
用途：描述普通版测试的评测目标、命中对象、分数区间和结果解释。  
状态：待人工补充与校对

## 1. 普通版定位

普通版用于快速鉴别用户在明清史观相关议题上的整体倾向，重点区分：

- 理性历史观
- 完全不了解历史
- 对历史了解不多，但轻度偏好明朝的萌萌人
- 已经切割的旧明粉
- 旧明粉
- 新明粉
- 朱元璋梦男

普通版不追求细分史观来源，只给出一个传播性强、容易截图分享的主结果。

## 2. 评测目标

普通版主要瞄准用户是否倾向以下判断：

- 为洪武型财政辩护
- 淡化禁钱、禁银、宝钞失败等货币财政问题
- 反感或拒绝「宋进明退」问题意识
- 倾向用情感身份维护明代制度
- 将明代制度问题归因于后人执行、外部环境或被抹黑
- 对强皇权、户籍束缚、海禁等制度成本不敏感

朱元璋梦男的主要特征
- 谈及纸币,激烈反对宋明纸币贬值速度对比,谈及美元贬值速度比大明宝钞快
- 否定洪武31年里190次起义,否定明朝开国67年里411次起义,频率烈度远高于宋代300多年411次农民起义
- 反对近代积贫积弱,朱元璋难辞其咎.
- 否认郑和下西洋被叫停是因为朱元璋的海禁政策.
- 明史被满清篡改了
- 反对朱元璋小农思维,像董志民,爱栓人
- 朱元璋的大明宝钞是第一款发行成功的没有贬值的纸币
- 明代铸币少是因为缺少铜矿,但是抛开明代276年云南没有大规模开发滇铜,直到孙可望时期才开开始大量铸币,清代大量开发滇铜不谈.

## 3. 不同类别的共同核心特点

### 理性历史观

资料状态：

- 真正的理性历史观资料暂缺。
- 当前 `samples/正常人/` 只能作为临时参考，不能直接等同于完整理性历史观样本。

核心特点：

- 评价王朝时优先看财政、货币、户籍、海禁、社会流动和国家能力等具体机制
- 能承认明代有历史处境和局部成就，但不会因此替洪武型财政、宝钞、海禁和强皇权自动辩护
- 面对宋明、明清比较时，愿意拆材料、论证和指标，不先按朝代身份站队
- 对「明史被满清篡改」「都是后人执行坏了」等护航话术警惕
- 对历史有较深认识，了解明代缺点、倒退和制度成本，不强烈偏向明朝

### 满遗

核心特点：

- 不是独立基础结果，而是「理性历史观」的 10% 抬旗彩蛋显示名
- 触发条件仍然是历史了解程度高、明朝偏向程度低或轻度偏明
- 结果页显示为「满遗」时，本质上仍按理性历史观解释
- 用途是增加娱乐性，模拟明粉语境中把批评洪武祖制的人直接抬旗

### 完全不了解历史

核心特点：

- 对多数议题停留在「听过一点」「不好判断」「看别人怎么说」
- 不一定有明显明粉倾向，也缺少稳定的制度史、财政史或公共历史判断框架
- 容易被题目中的熟悉名词带着走，例如洪武、宝钞、海禁、郑和下西洋
- 误判风险：不能因为不了解就直接判成理性历史观

### 对历史了解不多，但轻度偏好明朝的萌萌人

核心特点：

- 对明朝有朴素好感，常来自汉人王朝滤镜、人物魅力、影视游戏或历史同人表达
- 会觉得朱元璋、郑和、大明气象等符号有吸引力，但通常没有系统性洪武祖制辩护能力
- 面对财政、货币、户籍、海禁等问题时，容易选择「可能有道理，但我不太懂」
- 与「史同女性」样本可能有交集：更偏人物、关系、情感投射，而不是制度论证

### 已经切割的旧明粉

核心特点：

- 曾经或曾接近明粉身份，但开始主动退出、反思或切割
- 仍可能保留对明代部分人物、文化或历史情境的兴趣，但不再愿意被极端明粉绑定
- 能指出明粉圈层中的祖制护航、朱元璋神化、史料阴谋论或极端话术问题
- 面对宝钞、禁钱、海禁、强皇权等问题，更愿意承认负面影响

### 旧明粉

核心特点：

- 对明代有明确情感偏向，但未完全系统维护洪武祖制
- 愿意为明朝辩护，但不一定完全捆绑朱元璋、魏忠贤或强皇权
- 面对宝钞、禁钱、海禁、强皇权等问题时，可能有防御反应，但仍可能承认副作用
- 与已经切割的旧明粉不同，旧明粉仍保留更强明朝情感认同
- 与新明粉不同，旧明粉尚未形成完整制度护航体系

### 新明粉

核心特点：

- 比旧明粉更系统地维护明代制度，尤其倾向替洪武型财政、海禁、强皇权寻找合理性
- 常把明代问题归因于后人执行、外部环境、清代叙事、反明圈层或西方中心主义
- 对宋明比较、明清比较中的不利证据有较强防御反应
- 与封建保皇派样本高度重合，但尚未必然进入朱元璋梦男式强烈人格投射

### 朱元璋梦男

核心特点：

- 对朱元璋和洪武祖制有强烈人格化维护冲动
- 谈及纸币、起义频率、海禁、郑和下西洋、明史叙事等议题时，容易产生高强度防御
- 倾向否定或淡化大明宝钞失败、禁钱禁银、低货币化、基层控制、海禁等制度成本
- 可能完全否定袁崇焕、于谦、张居正等名臣贡献，对魏忠贤等形象产生同情或错位投射
- 常见话术包括「明史被满清篡改」「朱元璋的大明宝钞没有失败」「近代困境不能怪洪武」等

## 4. 题目设计原则

- 每题只测一个主要判断，不做复合陷阱题
- 选项从强认同到强反对逐级排列
- 题目要能区分立场强度，而不是考冷门知识
- 题目语气可以讽刺，但不能让用户一眼看出唯一正确答案
- 尽量覆盖财政、货币、宋明比较、海禁、强皇权、史观方法等主题

## 5. 评分规则

**与当前工程一致时**（`src/data/basic-test-config.ts`）：每次测验固定随机抽 **20** 题，其中 **6** 道核心题（权值 2）、**14** 道补充题（权值 1）；单轴理论满分 **78**（\(6 \times 2 \times 3 + 14 \times 1 \times 3\)）。

下文表 1、§6 区间原按 **72** 分制书写（对应旧稿「4 核心 + 16 补充」）；落在代码里时按 **×78/72** 换算为 78 分制边界（例如「49–72」历史高分段对应 **53–78**）。

---

**表 1（设计文档 / 100 题库口径，待与上线题库同步时校对）**

普通版题库可设计为共 100 题：

- 核心题型 20 题，权值 2
- 其他题型 80 题，权值 1
- 若采用「4 核心 + 16 补充」抽题：单轴满分 **72**

普通版使用两个评分维度：

- 历史了解程度：判断用户是了解历史，还是不了解历史
- 明朝偏向程度：判断用户的明粉程度

每题 4 个选项，默认双维度分值如下：

| 选项 | 历史了解程度 | 明朝偏向程度 | 含义 |
| --- | --- | --- | --- |
| A | 1 | 3 | 强烈认同明粉式判断，偏向明朝但分析能力不一定强 |
| B | 2 | 2 | 部分认同明代辩护，了解一些背景但仍有朝代偏好 |
| C | 0 | 1 | 中立、摇摆、不确定，主要用于识别不了解历史 |
| D | 3 | 0 | 反对明粉式判断或倾向制度分析，历史了解程度较高 |

单题得分为对应维度分值乘以题目权值。

每个维度每次测试满分均固定为 72 分：

```txt
核心题型：4 题 × 3 分 × 2 权值 = 24 分
其他题型：16 题 × 3 分 × 1 权值 = 48 分
单维度总分：72 分
```

## 7. 命中样本

### 理性历史观

典型观点：

- 具体问题具体比较，不用朝代身份替代制度分析
- 明初制度有历史背景，但长期成本需要正面讨论
- 论文和材料可以被质疑，但不能先按立场定罪
- 对明代有较深认识，了解明代缺点和倒退，但不强烈偏向明朝

容易选择的选项：

- 多数题倾向 D，少量题选择 B 或 C

### 满遗

典型观点：

- 与理性历史观相同
- 只是显示层被随机抬旗，不代表真的进入明粉或反明身份判断

容易选择的选项：

- 与理性历史观相同

### 完全不了解历史

典型观点：

- 我不太懂，先看别人怎么说
- 这个问题可能很复杂，我没有明确立场

容易选择的选项：

- 大量选择 C，少量选择 B 或 D

### 对历史了解不多，但轻度偏好明朝的萌萌人

典型观点：

- 明朝感觉挺有气势，朱元璋、郑和、大明这些符号很有吸引力
- 具体财政货币问题不了解，但直觉上不想把明朝说得太差

容易选择的选项：

- C 和 B 较多，遇到人物、王朝身份、海洋气象题可能选择 A

### 已经切割的旧明粉

典型观点：

- 以前偏明或接近明粉，现在发现很多说法站不住
- 喜欢明朝不等于维护朱元璋所有制度
- 极端明粉、朱元璋梦男或新明粉的话术需要切割

容易选择的选项：

- D 和 B 较多，遇到情感题可能仍选择 B

### 旧明粉

典型观点：

- 明朝值得喜欢，但朱元璋和洪武制度也有问题
- 袁崇焕、于谦、张居正等名臣仍有贡献，不能被一刀切否定
- 明代有问题，但不能因此全盘否定明代

容易选择的选项：

- B 较多，少量 A 和 C

### 新明粉

典型观点：

- 明代制度不是退步，很多问题是后人执行、外部环境或反明叙事造成的
- 宋明比较和明清比较经常带有立场偏见

容易选择的选项：

- A 和 B 较多，涉及洪武祖制、海禁、明清比较时更容易选择 A

### 朱元璋梦男

典型观点：

- 洪武祖制方向没错，问题主要是后人、清修明史或反明话术
- 大明宝钞、起义频率、海禁后果、近代困境等批评都需要强烈反驳

容易选择的选项：

- 大量选择 A，尤其核心题型高频命中

## 9. 后续校准

上线后可根据以下信息调整题目和区间：

- 各结果人数分布是否过度集中
- 用户反馈哪些题目太明显或太偏
- 是否大量用户集中在萌萌人或旧明粉区间
- 朱元璋梦男结果是否足够有传播性
- 理性历史观结果是否过于无聊
