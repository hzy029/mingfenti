## 2. 选项评分口径

每题 4 个选项。默认语气与后续代码评分一致：

| 选项 | 历史了解程度 | 明朝偏向程度 | 方向 |
| --- | ---: | ---: | --- |
| A | 0 | 3 | 强明粉 / 朱元璋梦男式判断 |
| B | 1 | 2 | 低了解但高明朝偏向 / 旧明粉式辩护 |
| C | 0 | 1 | 低了解、中等明朝偏向 / 教材式直觉 / 路人摇摆 |
| D | 3 | 0 | 倾向材料、机制和制度后果分析 |

## 3. 校对重点

- A/B/C 是否像真实答题者会选的自然回答，而不是模板化立场标签。
- 是否有题目立场太明显，让 D 看起来像唯一正确答案。
- 是否有题目过度依赖未核实数字或具体案例。
- 军事、行政题是否足够独立，不只是财政题换皮。
- 横向史观题是否仍能服务普通版结果，而不是滑向 Pro 四维。
- 史同、粉圈、教条主义题量是否克制。
