# Pro 版四维结果设计

版本：v0.1  
日期：2026-04-30

> **状态（2026-05）**：**已废止。** 当前产品中的 **Pro** 指 **双轴四选一** 测验，结果见 [`basic-results.ts`](../../src/data/basic-results.ts) 与 [`pro-evaluation-standard.md`](./pro-evaluation-standard.md)。本文仅保留历史草案，**不作验收依据**。

## 1. 四维

- 西方中心主义
- 教科书教条主义
- 封建主义
- 科学社会主义

## 2. 强度

每个维度归一化为 0-100 分。

| 强度 | 分数 | 含义 |
| --- | --- | --- |
| 无 | 0-34 | 该维度不明显 |
| 弱 | 35-64 | 该维度有一定倾向 |
| 强 | 65-100 | 该维度是主要史观成分 |

## 3. 维度到称号

| 维度 | 无 | 弱 | 强 |
| --- | --- | --- | --- |
| 西方中心主义 | 萌萌人 | 西方中心论者 | 西方中心论者 |
| 教科书教条主义 | 萌萌人 | 教科书主义者 | 朱家太监 |
| 封建主义 | 萌萌人 | 旧明粉 | 封建遗老 |
| 科学社会主义 | 萌萌人 | 正常人 | 正常人 |

## 4. 主结果优先级

1. 四维全无：萌萌人
2. 教科书教条主义强 + 封建主义强：朱家太监
3. 教科书教条主义强：朱家太监
4. 封建主义强：封建遗老
5. 科学社会主义强：正常人
6. 西方中心主义强：西方中心论者
7. 封建主义弱：旧明粉
8. 教科书教条主义弱：教科书主义者
9. 西方中心主义弱：西方中心论者
10. 科学社会主义弱：正常人

说明：优先级故意把「朱家太监」「封建遗老」放高，因为这两个更有传播性；即便用户同时有其他维度，只要封建和教条成分足够强，结果就会更有节目效果。

## 5. 称号文案

### 萌萌人

四维都没达标。你可能只是路过吃瓜，对明清史观大战尚未形成稳定偏见。恭喜，暂时还有救。

### 西方中心论者

你习惯用资本主义萌芽、海洋扩张、市场经济和近代化路径解释宋明差异。问题不大，但小心把所有王朝都塞进欧洲剧本。

### 教科书主义者

你比较依赖标准答案、教材结论和固定朝代评价模板。优点是稳定，缺点是容易把复杂历史问题背成填空题。

### 朱家太监

你对祖制、帝王叙事和官方秩序有超出常人的维护冲动。只要有人批评洪武制度，你的护驾机制就会自动启动。

### 旧明粉

你对明代有明显情感偏向，但尚未完全滑入祖制万岁的深水区。属于还可以交流，但需要随身携带参考文献的类型。

### 封建遗老

你对强皇权、户籍束缚、重农抑商和低流动社会抱有异常稳定的理解甚至同情。你不是在研究明朝，你是在想念秩序。

### 正常人

你更倾向用生产力、财政汲取、国家能力和社会形态演进来解释历史问题。恭喜，你暂时没有被单一王朝滤镜绑架。

## 6. 展示建议

结果页展示：

- 主称号
- 一句话锐评
- 四维条形图
- 每个维度的强度：无 / 弱 / 强
- 副标签：列出第二高维度，例如「副成分：教科书教条主义弱」
