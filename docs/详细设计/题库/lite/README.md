# 普通版判断题题库

本目录为**普通版（lite）判断题**的正式文档题库：题干、倾向字段与材料来源以 Markdown 维护，供产品与实现侧对齐题面、权重与判分规则。当前线上 lite 流程仍可能从代码内母题生成判断陈述；接入本目录时，应以此处为单一事实来源并同步 `src/lib` 与题型定义。

## 目录与阅读顺序

建议按下列顺序阅读或抽题编排（由「制度与史料」到「网络话术与样本」）：

1. [`00-reference-extract.md`](00-reference-extract.md) — 从 `docs/references` 提炼的可出题材料池，按财政、军事、行政、横向史观分组；写正式题干前须按条目中「核对要求」回源。
2. [`01-core-extreme.md`](01-core-extreme.md) — **核心极端题**（题头 `E###`），测明朝/明粉式辩护或攻击倾向。
3. [`02-core-rational.md`](02-core-rational.md) — **核心理性题**（题头 `R###`），测在批评明代问题的同时能否承认史实复杂性与明代实际面向。
4. [`ordinary/01-ordinary-extreme.md`](ordinary/01-ordinary-extreme.md) — **普通极端题**（题头 `OE###`），话题多为网络论战、汉服、弃地、奴变、左宗棠与史同话术等。
5. [`ordinary/02-ordinary-rational.md`](ordinary/02-ordinary-rational.md) — **普通理性题**（题头 `OR###`），与 `ordinary` 极端题按主题大致对应，便于同卷成组。

## 题量与权重约定

- 当前正文题量（随文件维护，以各文件 `##` 题头为准）：核心极端约 **44** 题（`E001`–`E044`），核心理性 **20** 题（`R001`–`R020`），普通极端 **15** 题（`OE001`–`OE015`），普通理性 **13** 题（`OR001`–`OR013`）。
- **核心题**（`E###` + `R###`）：接入测试时建议权重高于 **普通题**（`OE###` + `OR###`）。
- 题号前缀含义：`E` 核心极端，`R` 核心理性，`OE` 普通极端，`OR` 普通理性；序号在各前缀内从 `001` 连续递增，重排正文时须同步改题头。

## 极端题格式

每题使用固定字段：

```md
## E001

类型：极端题

核心题：是

题干：...

明粉倾向：支持/反对

材料来源：...
```

判分规则：用户选择与 `明粉倾向` 相同，增加明朝偏向分；否则增加理性分。`OE###` 字段相同，将 `E` 换为 `OE` 即可。

## 理性题格式

每题使用固定字段：

```md
## R001

类型：理性题

核心题：是

题干：...

理性倾向：支持/反对

材料来源：...
```

判分规则：用户选择与 `理性倾向` 相同，增加理性分；否则扣明朝偏向分。理性题用于识别用户是否能在批评明代问题的同时承认明朝的实际贡献与复杂性；明朝偏向分允许为负数，后续可用于其它结果维度。`OR###` 字段相同，将 `R` 换为 `OR` 即可（普通理性题中 `核心题` 可为「否」，表示题干为对常见非理性话术的否定陈述，仍以 `理性倾向` 对齐判分）。

## 出题原则

- 人工逐题生成，不用脚本批量制造题干。
- 每题只测试一个主要判断。
- 极端题主要测试攻击或鼓吹明朝的粉圈化倾向。
- 理性题主要测试是否能承认明朝贡献与制度复杂性，而不是把明朝写成一无是处。
- 样本类材料只作为话术来源，不作为事实依据。
- 修改 `docs/详细设计/题库/lite` 下四个题库 Markdown 后，运行 **`npm run generate:lite-bank`** 重新生成 [`src/data/lite-bank-data.ts`](../../../../src/data/lite-bank-data.ts) 并提交。

## 材料池与正文题块的关系

- `00-reference-extract.md` 中的小节顺序与题库主题线对齐：财政能力与大项支出在前，货币链条（宝钞、禁钱、白银、铸钱与区域）随后，再接军事、行政与横向史观。
- `02-core-rational.md` 中题目顺序按主题编排：大一统与史观方法、财政与跨朝比较、崇祯与辽东、海洋政策、基层册籍与户口、社会流动与起义、身份叙事与长期制度遗产。
- `ordinary` 下极端题与理性题按同一话题带排列：土木堡、汉服、弃地、奴变、新疆与海防塞防、史同与粉圈讨论。
