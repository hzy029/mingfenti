# 普通版判断题 · 普通题

本目录存放普通版判断题的非核心题，用于补充核心题之外的公共历史话题、粉圈话术和史观校准。

## 文件说明

- `01-ordinary-extreme.md`：普通极端题，测试攻击或鼓吹明朝的粉圈化倾向。
- `02-ordinary-rational.md`：普通理性题，测试能否承认历史复杂性与明朝相关贡献。

## 判分规则

- 极端题：用户选择与 `明粉倾向` 相同，增加明朝偏向分；否则增加理性分。
- 理性题：用户选择与 `理性倾向` 相同，增加理性分；否则扣明朝偏向分。
