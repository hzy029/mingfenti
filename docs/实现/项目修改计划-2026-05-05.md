# 2026-05-05 项目修改计划

本文档记录当日已确认的产品与工程变更范围，便于留档与验收。

## 1. 首页公告栏与首屏文案

- 弹窗「最新公告」与深蓝首屏引用统一替换为：
  - 大明无好百姓,却有好太监.
  - 一个人如果成为了明粉 那他就不是人了
- 涉及文件：`src/components/home-announcement-modal.tsx`、`src/app/page.tsx`。

## 2. 答题最短用时（写库阈值）

- 答题须知文案：不满 **20** 秒视为过快（原 30 秒）。
- API `POST /api/basic-attempts`：`MIN_RECORDED_DURATION_SECONDS` 调整为 20；低于阈值返回 `too-fast` 且不写库。
- 涉及文件：`src/components/test-intro-view.tsx`、`src/app/api/basic-attempts/route.ts`；README 与 handoff 等文档同步描述。

## 3. 结果页：客观中立 / 满遗彩蛋 + 留言板置顶概要

- 当 `resultId` 为 `objective-neutral` 或 `manchu-loyalist` 时，在「诊断结果」下方展示与首页同源的置顶主题 + 最新已发布帖摘要。
- 客户端通过 `GET /api/board/pin-preview` 拉取；可点击进入 `/board`。
- 数据语义为「置顶主题下最新帖预览」，非按点赞排序的「热评」。

## 4. 结果页 5 秒自动保存图片

- 进入有效结果页后延迟 5 秒调用与「保存图片」按钮相同的导出逻辑；单次触发，卸载时清理定时器。
- 注意：部分浏览器可能拦截无用户手势的下载，需真机验证。

## 5. QQ 内置浏览器首页提示

- 通过 `navigator.userAgent` 识别 QQ 内置浏览场景（排除微信 `MicroMessenger`），在首页展示可关闭提示条，建议用系统浏览器打开。

## 6. 结果横轴展示

- 标签「西史辨伪化程度」改为「被伪史论污染程度」。
- 该轴分数在页面与导出图中显示为负分（如 `-78 / -78`），**会话与上报仍保持 0–78**，仅展示层取负号。

## 7. 验收要点

- 首页文案与 QQ 提示；20 秒阈值与文档一致；结果页条件块与自动下载；横轴标签与负分在页面与 PNG 中一致。
