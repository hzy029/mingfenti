# 题库主题索引

版本：v0.1  
日期：2026-04-30

本索引用于把资料库内容映射到测试题库主题。后续生成题目时，优先按本索引取材。

## 当前样本分类

- `samples/正常人/`：理性历史观的临时近似样本，真正理性历史观资料暂缺
- `samples/已经切割的旧明粉/`：已经从明粉身份退出或开始反思的样本
- `samples/旧明粉/`：仍偏明但未系统护航洪武祖制的样本
- `samples/新明粉/`：更系统维护明代制度、对不利比较有防御性的样本
- `samples/明粉保皇派朱元璋梦男/`：强朱元璋人格投射、祖制护航、封建保皇派样本
- `samples/教条主义/`：否认宋明财政比较、科班/教材式话术，后续用于 Pro 教科书教条主义
- `samples/西方中心主义者/`：欧洲近代化、大航海、资本主义萌芽模板套用
- `samples/史同女性/`：历史同人女性向、CP 解读、饭圈化史观
