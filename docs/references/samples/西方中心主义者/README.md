# 西方中心主义样本

本目录用于存放 Pro 版「西方中心主义」维度的典型言论样本。

## 样本范围

- 用欧洲近代化路径直接套中国史
- 用资本主义萌芽解释宋明差异
- 把英国原始积累时代极其短视且血腥残暴的手段视为文明进步唯一的必由之路
- 用市场经济、城市商业、海外贸易作为单一评价标尺,
- 把中国历史发展放进欧洲剧本中解释的言论
- 把复杂的地理,制度性问题,变成一个简化的所谓思维问题,好像思维一解放,所有问题都迎刃而解了.

## 目标用途

- 提取「西方中心主义」维度的典型特征
- 区分合理比较与西方中心主义套模板
- 为 Pro 版题库设计提供真实话术样本
