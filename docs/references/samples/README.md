# 典型言论样本库

本目录用于存放明粉、朱元璋梦男等典型言论样本，供后续提取评测特征和设计题目使用。

## 内容范围

- 明粉典型辩护话术，即 Pro 版「封建主义」维度中的封建保皇派思想样本
- 朱元璋梦男典型言论
- 西方中心主义典型言论
- 史同女性相关典型言论
- 洪武型财政辩护样本
- 宋明比较防御样本
- 大明宝钞、禁钱、禁银相关话术
- 海禁、郑和下西洋相关话术
- 明史被篡改、满清抹黑等史料阴谋论样本

## 建议格式

可以按主题新建 Markdown 文件，例如：

```txt
feudal-royalist/hongwu-finance.md
feudal-royalist/paper-money.md
feudal-royalist/maritime-ban.md
feudal-royalist/historiography-conspiracy.md
western-centrism/capitalism-sprout.md
western-centrism/maritime-expansion.md
史同女性/character-projection.md
史同女性/fandom-style-history.md
```

单条样本建议格式：

```md
## 样本 001

原话：
> 这里粘贴典型言论。

来源：
- 平台：
- 链接：
- 时间：

初步标签：
- 洪武财政辩护
- 宝钞成功论

可提取特征：
- 将宝钞失败归因于后人执行
- 回避发行信用和兑换机制问题
```

## 使用原则

- 优先保存原话和来源，便于回看语境
- 提取题目时只概括特征，不直接大段搬运
- 对涉及个人攻击、隐私或平台敏感内容的样本，后续公开页面不要直接展示
