# 封建保皇派样本

本目录用于存放 Pro 版「封建主义」维度的典型言论样本，也就是明粉、朱元璋梦男一类言论。

## 样本范围

- 朱元璋梦男言论
- 洪武祖制辩护
- 强皇权、重典治国、户籍束缚辩护
- 大明宝钞成功论
- 禁钱、禁银、低货币化政策辩护
- 明史被篡改、满清抹黑等祖制护航话术
- 面对不利的话题答非所问
- 遇事不决,拐满清,说清朝更差
- 或者扣帽子,人身攻击,攻击ip地址比如是东北人ip,说是满人,给抬旗,鹿角文化,妈妈窝
- 热衷于造谣,发明伪史

## 目标用途

- 提取「封建主义」维度的典型特征
- 提取普通版「朱元璋梦男」结果特征
- 为后续正式题库提供真实话术样本
