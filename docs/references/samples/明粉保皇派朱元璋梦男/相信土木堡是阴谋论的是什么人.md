土木堡⚲在欧洲。这玩意不是信仰，谈不上信不信。这玩意属于科学，因为它可证伪。

公元前334年，亚历山大开始东征，从马其顿都城佩拉⚲（今希腊北部）到东方最远抵达点比亚斯河⚲（印度河支流），直线距离约4,800 公里，行军征战的实际距离超过10,000 公里。

以3万步兵，5千骑兵，耗时10年完成东征。

补给方式：以战养战 + 建立补给链 + 海运优先。

过了一千五百六十八年。

于公元1235年蒙古第二次 "长子西征⚲"，蒙古军队从蒙古高原到欧洲最远抵达点（亚得里亚海沿岸，今克罗地亚一带），直线距离约5,500 公里，实际征战行军距离超过12,000 公里。

以12万蒙古骑兵，数万攻城部队，耗时7年完成西征。

补给方式：移动畜牧 + 因粮于敌 + 奥鲁后勤。

又过了二百零七年。

于公元1449年，明英宗⚲西征欧洲土木堡。从北京到欧洲土木堡（德国莱茵河畔卡尔斯鲁厄附近的图尔姆山）的直线距离7872公里。

陆路行军路线：北京出发→经张家口→过长城→沿河西走廊→新疆→中亚 (哈萨克斯坦、乌兹别克斯坦)→里海→高加索→黑海→东欧→德国→卡尔斯鲁厄→图尔姆山，实际征战行军距离12000 公里。明军骑兵需250-300 天可抵达图尔姆山。

补给方式：移动畜牧 + 因粮于敌 + 奥鲁后勤。



海路行军路线：天津港→印度洋→红海 / 地中海→大西洋→北海→鹿特丹港（约 14800 公里）+ 鹿特丹→卡尔斯鲁厄莱茵河内河航运（约 800 公里），明军需180-240 天可抵达图尔姆山。

补给方式：以战养战 + 建立补给链 + 海运优先。



明英宗以骑军、步军、海军、后勤辎重部队共20万人，沿陆路、海路双线进攻。从公元1449年开始西征，到公元1457年回到北京复辟重新执政，期间共8年时间。

土木堡在欧洲，并非不可能。

在明英宗之前有郑和下西洋，大明的航海技术与运力，肯定比一千七百多年前的亚力山大要强不少。明英宗的骑兵主力是蒙古诸部归附的三千营⚲，战术素养继承于蒙古军队。

所以，亚力山大的军队乘船可以用10年时间东征万里，蒙古军队骑马可以用7年时间西征万里，那么明英宗的军队骑马与乘船，用8年时间西征万里也是可能的。
