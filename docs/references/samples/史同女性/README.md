# 史同女性样本

本目录用于存放「史同女性」相关的典型言论样本，供后续判断是否需要扩展普通版结果、Pro 版维度或结果页锐评素材。

## 样本范围

- 历史同人女性向创作中的典型角色投射
- 对历史人物关系、人格、情感和权力结构的二创式解读
- 将历史讨论转化为 CP、梦向、嬷向、饭圈化表达的话术
- 有历史情感偏好,但是真的会去看史料比朱元璋梦男强一万倍
- 与明清人物、宋明比较、朱元璋形象相关的女性向同人表达

## 目标用途

- 判断「史同女性」是否需要成为独立样本类型
- 为结果页锐评、公告文案和题目语气提供素材
- 区分普通历史同人创作与参与史观争论的话术
- 避免把单纯娱乐创作误判成明粉、朱元璋梦男或西方中心主义
