# 资料库目录

本目录用于存放题库设计和文案校对所需的参考资料。

## 目录说明

- `papers/`：论文、PDF、CAJ、书摘、学术资料
- `本人内容/`：站主个人内容、原创文章、专栏稿、观点素材、阅读笔记
- `zhihu/`：搬运或整理的其他人知乎回答
- `samples/`：明粉、朱元璋梦男、西方中心主义等典型言论样本
  - `samples/正常人/`：理性历史观的临时近似样本，资料仍不足
  - `samples/已经切割的旧明粉/`：从明粉身份退出或反思的样本
  - `samples/旧明粉/`：仍偏明但未系统护航洪武祖制的样本
  - `samples/新明粉/`：更系统维护明代制度的样本
  - `samples/明粉保皇派朱元璋梦男/`：封建保皇派/明粉/朱元璋梦男言论
  - `samples/教条主义/`：后续 Pro 版教科书教条主义素材
  - `samples/西方中心主义者/`：西方中心主义言论
  - `samples/史同女性/`：历史同人女性向、CP 解读、饭圈化史观等言论
- `bibliography.md`：资料总目录
- `topic-index.md`：按题库主题整理的资料索引

## 使用建议

- 文件名尽量包含作者、年份和主题，例如：`刘光临-2009-宋明间国民收入长期变动之蠡测.pdf`
- 对重要资料可在 `notes/` 中写一份对应摘记
- 后续生成题目时优先使用本目录中的资料和人工校对内容
