# 资料库特征提取汇总

版本：v0.1  
日期：2026-05-01  
用途：后续正式普通版题库生成的特征库。当前只做概括提取，不直接生成题目。

## 1. 总体说明

当前 [`src/data/basic-questions.ts`](../../src/data/basic-questions.ts) 与代码中的题库仅用于流程测试，不代表最终题库。正式题库应优先依据本文档、[详细设计/basic-evaluation-standard.md](../详细设计/basic-evaluation-standard.md) 与样本库继续生成。

普通版主框架：

- 历史了解程度
- 明朝偏向程度

普通版结果目标：

- 理性历史观
- 完全不了解历史
- 对历史了解不多，但轻度偏好明朝的萌萌人
- 已经切割的旧明粉
- 旧明粉
- 新明粉
- 朱元璋梦男

说明：

- `满遗` 不是独立评测类别，只是 `理性历史观` 的 10% 彩蛋显示名。
- `samples/正常人/` 目前只作为理性历史观的临时参考样本。真正的理性历史观资料仍暂缺，不能从现有样本过度外推。
- 本轮暂不修订 Pro 四维算法；`samples/教条主义/` 仅登记为后续 Pro 版“教科书教条主义”维度素材来源。

## 3. 理性历史观

资料状态：

- 真正的理性历史观资料暂缺。
- `samples/正常人/` 可作为临时参考，但不能完全代表此类人群。

核心特征：

- 对历史有较深认识，能理解明代的缺点、倒退与制度成本。
- 不强烈偏向明朝，不把朝代身份当作制度评价前提。
- 面对宋明、明清比较时，愿意拆解财政、货币、户籍、海禁、社会流动、国家能力等具体指标。
- 能承认明代存在历史处境、局部成就和具体人物贡献，但不会因此替洪武型财政、宝钞、禁钱、海禁、强皇权自动辩护。
- 对“明史被满清篡改”“后人执行坏了”“都是反明叙事”等护航话术保持警惕。

典型话术：

- “具体问题具体比较，不能用朝代好恶代替制度分析。”
- “明初有恢复秩序的一面，但货币、财政、户籍和海禁成本也要正面讨论。”
- “论文可以质疑，但要拆材料和方法，不能先按立场定罪。”

可测特征：

- 是否接受宋明财政、货币、市场规模比较的意义。
- 是否愿意承认明代具体制度存在长期负面影响。
- 是否能区分人物好感、王朝情感与制度评价。

误判边界：

- 不能把所有批评明代的人都判成“理性历史观”。
- 不能把不了解历史、只是跟着反明情绪走的人判成“理性历史观”。

## 4. 完全不了解历史

资料状态：

- 可从普通版标准中定义，样本目录中未形成专门材料。

核心特征：

- 对多数议题停留在“听过一点”“不好判断”“看别人怎么说”。
- 缺少稳定的制度史、财政史、货币史或公共历史判断框架。
- 容易被熟悉名词带着走，例如洪武、宝钞、郑和下西洋、明史、宋明比较。
- 不一定偏明，也不一定反明。

可测特征：

- 高频选择“不确定”“不了解”“看情况”。
- 对基础概念缺少判断，例如宝钞、禁钱、白银化、海禁、户籍迁徙。

误判边界：

- 不能因为不偏明就判成理性历史观。
- 不能因为不了解而直接判成朱元璋梦男。

## 5. 轻度偏明萌萌人

参考目录：

- `samples/史同女性/`

核心特征：

- 对明朝有朴素好感，常来自人物魅力、汉人王朝滤镜、影视游戏、历史同人、CP 或梦向表达。
- 对朱元璋、马皇后、朱标、张居正等人物关系或形象有兴趣。
- 更偏人物、关系、情感投射，不一定有制度论证能力。
- 面对财政、货币、户籍、海禁等议题，常表现为“可能有道理，但我不太懂”。

典型话术：

- “这个人物很有魅力。”
- “我喜欢这个朝代氛围，但具体制度不太懂。”
- “人物关系比财政问题更能吸引我。”

可测特征：

- 是否因人物魅力或情感投射产生轻度明朝偏好。
- 是否缺少对洪武财政、宝钞、海禁等制度问题的稳定判断。

误判边界：

- 不能把普通历史同人创作直接判成朱元璋梦男。
- 不能把人物厨、CP 解读、梦向表达直接等同于制度护航。

## 6. 已经切割的旧明粉

参考目录：

- `samples/已经切割的旧明粉/`

核心特征：

- 曾经或曾接近明粉身份，但开始主动退出、反思或切割。
- 能意识到明粉圈层中的祖制护航、朱元璋神化、史料阴谋论或极端话术存在问题。
- 仍可能保留对明代部分人物、文化或历史情境的兴趣，但不再愿意被极端明粉绑定。
- 对洪武财政、宝钞、海禁、强皇权等问题更愿意承认负面影响。

典型话术：

- “以前偏明，现在觉得很多说法站不住。”
- “喜欢明朝不等于要护朱元璋所有制度。”
- “被更极端的明粉切割，反而发现自己没那么纯。”

可测特征：

- 是否承认自己曾经有明朝偏好。
- 是否能主动指出明粉话术中的问题。
- 是否愿意与朱元璋梦男、新明粉做身份切割。

误判边界：

- 不应判成完全不了解历史。
- 不应判成新明粉或朱元璋梦男。

## 7. 旧明粉

参考目录：

- `samples/旧明粉/`

核心特征：

- 仍偏爱明朝，有较明确的明代情感认同。
- 可能认可袁崇焕、于谦、张居正等明代名臣贡献。
- 愿意为明朝辩护，但还没有系统维护洪武祖制、魏忠贤、强皇权或海禁。
- 面对洪武财政、宝钞、禁钱、海禁等问题，可能有防御反应，但仍能承认部分副作用。

典型话术：

- “明朝有问题，但不能全盘否定。”
- “喜欢明朝，不代表所有制度都没问题。”
- “有些批评太过了，但朱元璋也不是没有问题。”

可测特征：

- 明朝情感偏好较高。
- 对制度问题有一定辩护，但仍保留讨论空间。
- 面对不利材料不会立刻诉诸“满清篡改”或强烈护朱。

误判边界：

- 不应与已经切割的旧明粉混同：旧明粉仍保留更强明朝情感认同。
- 不应与新明粉混同：旧明粉未必系统维护明代制度。

## 8. 新明粉

参考目录：

- `samples/新明粉/`

核心特征：

- 比旧明粉更系统地维护明代制度和明代叙事。
- 倾向替洪武型财政、海禁、强皇权、禁钱禁银等寻找合理性。
- 常把明代问题归因于后人执行、外部环境、清代叙事、反明圈层或西方中心主义。
- 对宋明财政、货币、市场规模、明清比较中的不利证据有较强防御反应。
- 与朱元璋梦男有重叠，但不一定形成强烈朱元璋人格投射。

典型话术：

- “明代不是退步，是被误读了。”
- “很多问题是后人执行坏了，不是祖制方向错。”
- “宋明比较本身就带有反明立场。”

可测特征：

- 是否系统维护明代制度。
- 是否用外部归因化解制度批评。
- 是否对宋明、明清不利比较有稳定防御。

误判边界：

- 不应把所有新明粉判成朱元璋梦男。
- 只有出现强烈朱元璋人格投射和祖制护航时，才进入朱元璋梦男。

## 9. 朱元璋梦男

参考目录：

- `samples/明粉保皇派朱元璋梦男/`

核心特征：

- 对朱元璋和洪武祖制有强烈人格化维护冲动。
- 将批评洪武财政、宝钞、海禁、起义频率、强皇权等议题视为对朱元璋或大明的攻击。
- 面对不利材料时，容易答非所问、转移到清朝更差、扣帽子、人身攻击、抬旗、IP 地域攻击等。
- 倾向使用“明史被满清篡改”“反明话术”“大明宝钞没有失败”“近代困境不能怪洪武”等辩护。
- 可能发明伪史或使用未经验证的历史叙事护航。

典型话术：

- “大明宝钞是成功纸币，失败是后人污蔑。”
- “近代积贫积弱不能怪朱元璋。”
- “明史被满清篡改了。”
- “批评洪武制度就是明黑/满遗/反明。”

可测特征：

- 是否强烈否认大明宝钞失败或制度性贬值问题。
- 是否否定洪武和明初起义频率、烈度比较。
- 是否否认郑和下西洋被叫停与海禁、废海改漕相关。
- 是否把制度批评转化为身份攻击。

误判边界：

- 不应把旧明粉、新明粉中仍有讨论空间的人直接判为朱元璋梦男。
- 朱元璋梦男需要同时具备高明朝偏向和强朱元璋/洪武祖制护航。

## 10. 教条主义

参考目录：

- `samples/教条主义/`

本轮用途：

- 仅登记为后续 Pro 版“教科书教条主义”素材来源。
- 暂不修订 Pro 四维评分规则。

核心特征：

- 否认宋明财政比较的意义，认为不同朝代财政、货币体系不可比较。
- 以“科班”“教材”“学术规范”“不能这么比”等姿态拒绝具体指标比较。
- 默认白银单一货币体系比金银铜铁纸复合体系先进。
- 对宋明财政、货币、纸币、铜钱、白银体系的比较采取先验排斥态度。

典型话术：

- “宋明财政不能这么比较。”
- “不同历史时期没有可比性。”
- “白银体系当然更先进。”
- “你这不是科班的比较方法。”

可测特征：

- 是否拒绝比较本身，而不是指出具体比较方法的问题。
- 是否把学术规范当作阻断讨论的权威话术。
- 是否默认某种货币形态线性先进。

## 11. 西方中心主义者

参考目录：

- `samples/西方中心主义者/`

本轮用途：

- 记录为后续 Pro 版“西方中心主义”维度素材。
- 暂不修订 Pro 四维评分规则。

核心特征：

- 用欧洲近代化路径直接套中国史。
- 用资本主义萌芽、大航海、殖民扩张、市场经济、城市商业作为单一进步尺度。
- 把复杂的地理、制度、财政、技术条件简化为“思维是否开放”。
- 容易把英国原始积累或海洋扩张视为文明进步的必由之路。

可测特征：

- 是否用欧洲剧本解释宋明差异。
- 是否把大航海、海外贸易、资本主义萌芽当作唯一进步方向。
- 是否忽略具体制度和资源条件。

## 12. 史同女性

参考目录：

- `samples/史同女性/`

核心特征：

- 更关注人物关系、情感、人格、CP、梦向、饭圈化表达。
- 可能有明朝人物偏好，但通常不是制度护航型明粉。
- 有历史情感偏好，也可能会查史料，比纯朱元璋梦男更有辨析空间。
- 样本可用于区分“轻度偏明萌萌人”和“制度护航型明粉”。

可测特征：

- 是否主要因人物魅力、关系投射或同人表达产生偏好。
- 是否缺少对财政、货币、海禁、强皇权等制度问题的系统辩护。

误判边界：

- 不应把所有历史同人女性向表达判成明粉或朱元璋梦男。
- 只有当人物偏好转化为洪武祖制护航，才可能进入新明粉或朱元璋梦男。
